## Customization Options

This curriculum is designed to be flexible:

### Duration Variants
- **30-min Express**: Parts 1-2 only (authentication + first API call)
- **60-min Standard**: Full curriculum (recommended)
- **90-min Extended**: Add advanced topics, more hands-on time
- **2-hour Workshop**: Include extended Q&A and office hours

### Audience Adaptations
- **Beginners**: Slower pace, more explanation, pre-built code
- **Advanced**: Skip basics, focus on Ace API and complex integrations
- **Specific Language**: Focus examples on Python, JavaScript, or other
- **No AI**: Remove vibe coding elements, traditional API tutorial

### Event Types
- **Virtual**: Zoom/Teams, cloud IDEs, async components
- **In-Person**: Faster debugging, more interaction, pair programming
- **Hybrid**: Breakout rooms, dedicated online support
- **Async/Self-Paced**: Video modules, automated checks, forum support
