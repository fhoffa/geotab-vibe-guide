## Creating the Slides

### Tools
- **PowerPoint/Keynote**: Traditional presentation software
- **Google Slides**: Collaborative, cloud-based
- **Canva**: Modern, design-friendly templates
- **reveal.js**: HTML-based presentations (for developers)

### Design Guidelines
- Use Geotab brand colors (if available)
- Large, readable code snippets
- Screenshots of actual API responses
- Animated GIFs of coding in action
- Minimal text, maximum visuals
- Dark theme (developer-friendly)

### Best Practices
- Keep slides simple and uncluttered
- Use code syntax highlighting
- Include QR codes for links
- Number slides for easy reference
- Test on projector/large screen
- Export as PDF backup
