# Geotab Vibe Coding - Code Examples

This directory contains starter templates and working examples for the tutorial.

## Structure

```
examples/
├── python/               # Python examples using mygeotab library
├── addins/               # Geotab Add-In examples (custom MyGeotab pages)
└── instructions_for_ai/  # Prompts and instructions for AI assistants
```

## Available Examples

### Python (`python/`)
- ✅ Authentication examples
- ✅ Fetch data from Geotab API
- ✅ CLI dashboard
- ✅ Web dashboard with Streamlit
- ✅ Geotab Ace integration

See [python/](./python/) for detailed examples and tasks.

### Add-Ins (`addins/`)
- ✅ **Simple Dashboard** - Complete Add-In template with vehicle statistics
- More examples coming soon!

See [addins/](./addins/) for Add-In examples and [guides/GEOTAB_ADDINS.md](../guides/GEOTAB_ADDINS.md) for the full tutorial.

## Quick Start

Once examples are added, each subdirectory will have:

1. **README.md** - Setup and usage instructions
2. **requirements.txt** or **package.json** - Dependencies
3. **.env.example** - Template for credentials
4. **Source code** - Working example code

## Contributing

Have a great example? Submit a PR!

Requirements:
- Well-commented code
- README with setup instructions
- .env.example for credentials (never commit secrets!)
- Follows tutorial concepts
